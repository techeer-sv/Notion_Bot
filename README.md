# 🌠 슬랙봇 기반 블로그 자동화 도구
- 블로그 URL과 제목을 입력하면 Notion 데이터베이스에 자동 업로드
- 매주 전주의 게시물을 슬랙 채널에 자동으로 공유

<br/>

# 📑 Feature[1] : 블로그 등록
등록된 채널에서 SlackBot의 [제출] 명령어로 제목과 블로그 URL을 입력하면 테크 블로깅 챌린지의 Notion 데이터베이스에 자동으로 등록됩니다
